## Usage <br/><br/>

### Variants
Using multiple button variants allows for distinction between actions of higher and lower prominence.<br/><br/>

#### Contained
For higher prominence
```js
variant = 'contained'
```
<Preview>
  <Story name='contained'>
    <Box flex row jc='space-evenly'>
      <Button variant='contained'>Contained</Button>
    </Box>
  </Story>
</Preview>

#### Outlined
For less prominence
```js
variant = 'outlined'
```
<Preview>
  <Story name='outlined'>
    <Box flex row jc='space-evenly'>
      <Button variant='outlined'>Outlined</Button>
    </Box>
  </Story>
</Preview>

### Color
Apply color dynamically through your own brand theme.

Using the `brand` prop
```js
brand = 'primary' || 'secondary' || 'danger'
```
<Preview>
  <Story name='color'>
    <Box flex row jc='space-evenly'>
      <Button brand='primary'>Primary</Button>
      <Button brand='secondary'>Secondary</Button>
      <Button brand='danger'>Danger</Button>
    </Box>  
  </Story>
</Preview>

Or with the natural `style` attribute
```js
style={{ color: 'white', backgroundColor: 'blue' }}
```

### Sizes
Control the size of buttons
```js
size = 'sm' || 'md' || 'lg'
```
<Preview>
  <Story name='size'>
    <Box flex row jc='space-evenly'>
      <Button size='sm'>Small</Button>
      <Button>Default</Button>
      <Button size='lg'>Large</Button>
    </Box>
  </Story>
</Preview>

### Rounded
Apply round corners
```js
isRound = true
```
<Preview>
  <Story name='round'>
    <Box flex row jc='space-evenly'>  
      <Button isRound variant='contained'>Contained</Button>
      <Button isRound variant='outlined'>Outlined</Button>
    </Box>
  </Story>
</Preview>

### State
Provide visual feedback as cues to the state of a button
```js
isLoading = true
isDisabled = true
```
<Preview>
